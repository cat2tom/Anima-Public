Detailed Description

Inequality function for NLOPT estimation, helps maintain weights in reasonable bounds (i.e. {i=0}^N w_i = 1 , which is equivalent to {i=1}^N w_i <= 1)

param an MCM structure (to get the actual number of optimized weights)
